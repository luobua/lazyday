# Portal Credentials (AppKey 管理) 页面设计

> 覆盖 MASTER.md 对应部分，未提及的规则继续遵循 MASTER。

## 页面定位

开发者管理 API 凭证的核心页面。需要高效展示 AppKey 列表、安全地处理 SecretKey 一次性展示、支持 CRUD + 轮换操作。

## 表格设计

### 列定义

| 列 | 宽度 | 内容 |
|---|------|------|
| 名称 | auto | 名称（bold）+ 状态 Tag（活跃=green, 已禁用=red） |
| AppKey | 240px | 脱敏展示 `ak_pr****4b5c` + 复制按钮（CopyOutlined） |
| 权限范围 | auto | Tag 列表，蓝色 Tag |
| 创建时间 | 180px | 格式化日期 |
| 操作 | 260px | 详情 / 禁用(启用) / 轮换 / 删除 |

### 行为

- **hover 行**: 背景色 `#334155`（Slate 700）
- **空状态**: `<Empty description="还没有 AppKey" />`，下方 "创建第一个 AppKey" 链接
- **移动端**: `overflow-x: auto` 允许横向滚动

## 操作按钮

| 操作 | 类型 | 颜色 | 确认 |
|------|------|------|------|
| 详情 | `type="link"` + EyeOutlined | 默认蓝色 | 无（打开 Modal） |
| 禁用 | `type="link"` + StopOutlined | `danger` 红色 | `Popconfirm` "确定禁用此 AppKey？禁用后使用该 Key 的请求将被拒绝" |
| 启用 | `type="link"` + PlayCircleOutlined | 绿色 `#22C55E` | 无 |
| 轮换密钥 | `type="link"` + ReloadOutlined | 默认 | `Popconfirm` "确定轮换密钥？旧密钥将在 24 小时后失效" |
| 删除 | `type="link"` + DeleteOutlined | `danger` 红色 | `Popconfirm` "确定删除此 AppKey？删除后不可恢复" |

## SecretKey 一次性展示 Modal（关键交互）

创建成功后弹出：

```
┌──────────────────────────────────────────────┐
│  ⚠️ 请妥善保存 SecretKey                 [X]  │
├──────────────────────────────────────────────┤
│                                              │
│  AppKey                                      │
│  ┌────────────────────────────────────┐     │
│  │ ak_prod_8f3a2b1c...     [复制]     │     │ ← copyable
│  └────────────────────────────────────┘     │
│                                              │
│  SecretKey                                   │
│  ┌────────────────────────────────────┐     │
│  │ sk_live_a1b2c3d4e5...   [复制]     │     │ ← Fira Code 字体
│  └────────────────────────────────────┘     │
│                                              │
│  ⚠ SecretKey 仅此一次展示，关闭后无法再查看。  │
│    请立即复制并安全存储。                      │
│                                              │
│                          [我已保存，关闭]     │
└──────────────────────────────────────────────┘
```

### 关键 UX 规则

- 警告文字: `<Alert type="warning">`，醒目展示
- Key 展示: `<Typography.Text code copyable>`，使用 Fira Code 等宽字体
- **关闭前确认**: 点击关闭或 ESC 时弹出 `Modal.confirm("确定已保存 SecretKey？关闭后无法再次查看")`
- **禁止遮罩关闭**: `maskClosable={false}`
- 复制成功: `message.success('已复制到剪贴板')`，2s 消失

## 交互流程汇总

| 操作 | 流程 |
|------|------|
| 创建 | 填表 → loading → 成功弹出 SecretKey Modal → 列表刷新 |
| 禁用 | Popconfirm → loading → `message.success('已禁用')` → 状态 Tag 更新 |
| 启用 | 点击 → loading → `message.success('已启用')` → 状态 Tag 更新 |
| 轮换 | Popconfirm → loading → 弹出新 SecretKey Modal → 列表刷新 |
| 删除 | Popconfirm → loading → `message.success('已删除')` → 行移除 |

## 可访问性要点

- 表格操作按钮均有 `aria-label="禁用 生产环境"` 含 AppKey 名称
- Popconfirm 自动管理焦点
- SecretKey Modal: `aria-label="SecretKey 展示对话框"`
- 复制按钮: `aria-label="复制 AppKey"`
