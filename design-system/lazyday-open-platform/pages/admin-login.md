## 页面定位

平台管理员专用登录入口。需要与 Portal Login 视觉区分，强调管理身份。无注册入口（Admin 账号由后台创建）。

## 与 Portal Login 的差异

| 项目 | Portal Login | Admin Login |
|------|-------------|-------------|
| 标题 | "Lazyday 开放平台" | "管理控制台 / Admin Console" |
| 卡片宽度 | 420px | 400px（更紧凑） |
| 记住我 | 有 | **无**（安全考虑） |
| 忘记密码 | 有 | **无**（联系超管重置） |
| 注册链接 | 有 | **无**（后台创建） |
| 卡片顶部 | 无 | 可选：细微的顶部色条 `border-top: 3px solid #F59E0B`（警告黄，暗示管理权限） |

## 表单字段

| 字段 | 类型 | 校验 |
|------|------|------|
| 用户名 | `<Input prefix={<UserOutlined />}>` | 必填 |
| 密码 | `<Input.Password prefix={<LockOutlined />}>` | 必填 |

## 交互流程

```
用户输入 → 点击登录
  ├── Button loading=true
  ├── POST /api/admin/v1/auth/login
  ├── 成功
  │   ├── role === PLATFORM_ADMIN → router.push('/admin/overview')
  │   └── role !== PLATFORM_ADMIN → message.error('无管理员权限')
  └── 失败
       ├── 401 → message.error('用户名或密码错误')
       ├── 403 → message.error('该账号无管理员权限')
       └── 其他 → message.error(error.message)
```

## 错误状态

| 场景 | 展示 |
|------|------|
| 字段为空 | 内联红色提示 |
| 认证失败 | `message.error('用户名或密码错误')` |
| 权限不足 | `message.error('该账号无管理员权限')` — 区别于认证失败 |
| 网络错误 | `message.error('网络异常，请稍后重试')` |

## 可访问性

- Tab 顺序: 用户名 → 密码 → 登录
- Enter 键触发登录
- 精简页面，无多余干扰元素
