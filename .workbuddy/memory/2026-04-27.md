## Portal 菜单导航不切换问题
- **现象**：点击侧边栏菜单（除 /logs 外）URL 变了但页面内容不切换
- **根因**：Next.js App Router soft navigation 下，同一 layout 的 children 在 RSC 缓存存在时不会重新挂载
- **修复**：`<Content key={pathname}>` — 用 pathname 作为 key 强制 React 销毁重建子树
- **影响文件**：`portal/app/(dashboard)/layout.tsx`、`admin/app/(dashboard)/layout.tsx`
